# Tweeter - CyberArk-U_Code

## Overview

Tweeter project - Create a basic tweeter page with a few options: add posts/comments, delete posts/comments and get like/unlike to posts/comments.

## Technologies

1. JavaScript
2. HTML
3. CSS
